01 — Sobre

Curiosidade analítica com mão de quem constrói.

Sou Mateus Gabriel Reis, analista de dados e desenvolvedor de Franca-SP. Comecei no backoffice do Magazine Luiza, fui promovido para análise de negócios dentro do grupo Magalu e hoje atuo com Parcerias e Dados na aiqfome — sempre com a mesma obsessão: tirar clareza de dados bagunçados.

No dia a dia construo modelos em SQL / BigQuery, dashboards em Looker Studio e Power BI, e automações em Python e Apps Script. Quando o problema pede produto, viro a chave pro full-stack.

Essa mistura vem de uma formação multidisciplinar — radiologia, psicologia e agora Desenvolvimento de Software Multiplataforma na Fatec Franca. Aprendi a olhar um problema por ângulos diferentes antes de escrever a primeira linha.

"Nenhum dashboard se constrói sozinho — mas vale construir cada um como se fosse o único."
